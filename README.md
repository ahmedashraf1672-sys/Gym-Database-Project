# 🏋️ Gym Database Management System

A Gym Database Management System developed using SQL to efficiently manage gym operations. The database is designed to organize customer information, memberships, workout classes, employees, managers, and gym branches while maintaining data integrity through relational database design.

---

## 📌 Project Overview

This project demonstrates the design and implementation of a relational database system for a gym. It includes the complete database design process, from the Entity-Relationship Diagram (ERD) and Relational Schema to the SQL implementation.

---

## ✨ Features

- 👤 Manage customer information
- 💳 Track gym memberships and subscription periods
- 🏋️ Organize workout classes
- 📝 Enroll customers in workout classes
- 👨‍💼 Manage employees and managers
- 🏢 Store gym branch information
- 🔗 Maintain relationships using Primary and Foreign Keys

---

## 🗂️ Database Entities

- Customer
